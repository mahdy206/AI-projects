#  AI & Machine Learning Projects

A collection of AI and Machine Learning projects built during my university studies, covering computer vision, recommendation systems, and multimodal AI applications.

---

##  Projects

### 1. [Plant Disease Classification](./plant-disease-classification/)
> TensorFlow · MobileNetV2 · Transfer Learning · TFLite

A deep learning model that classifies 38 plant disease conditions across 14 crop types using MobileNetV2 transfer learning. Achieves ~94% validation accuracy and is exported to TFLite for mobile deployment.

### 2. [Movie Recommender System](./movie-recommender/)
> Python · GUI

A movie recommendation system with a graphical user interface for personalized movie suggestions.

### 3. [AI Fashion Chatbot](./ai-fashion-chatbot/)
> CNN · RNN · Computer Vision

An AI-powered fashion assistant that takes a photo of any clothing item and returns visually similar results using CNN for feature extraction and RNN for processing.
