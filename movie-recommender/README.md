# 🎬 Movie Recommender System

A movie recommendation system with a graphical user interface that provides personalized movie suggestions based on user preferences.

---

## ✨ Features

- 🎯 **Personalized Recommendations** — Suggests movies based on user input
- 🖥️ **Graphical Interface** — Easy-to-use GUI for interacting with the system
- ⚡ **Fast Results** — Quick recommendation generation

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| Language | Python |
| GUI | Tkinter / CustomTkinter |
| Core Logic | Collaborative / Content-based Filtering |

---

## 🚀 Getting Started

```bash
pip install -r requirements.txt
python movie_recommender1_gui.py
```
